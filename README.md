# 三家庭旅游记账

一个可直接在浏览器打开的三家庭旅行记账 app。它支持添加账单、自定义类别、本地保存、总额统计和三家均摊后的平账建议。

## 使用方式

直接打开 `index.html` 即可使用。如果要用本地预览地址，可以在这个目录启动任意静态文件服务。

## 功能

- 三个家庭固定为 `乐家`、`祺家`、`旦家`
- 预设类别：`交通`、`住宿`、`餐饮`、`门票`、`购物`、`其他`
- 支持新增自定义类别
- 支持金额、付款家庭、类别、日期、备注入账
- 自动统计总支出、每家均摊金额、每家已付金额、类别汇总
- 自动生成平账建议
- 使用浏览器本地保存数据
- 使用系统字体，适合直接发布到静态网页托管

## 多人云账本

项目已经预留 Supabase 云同步能力。启用步骤：

1. 在 Supabase 的 SQL Editor 里复制并运行 `supabase-schema.sql`。
2. 打开 Supabase 项目的 Project Settings → API Keys，复制 `anon public` key。
3. 把 `app.js` 顶部的 `SUPABASE_ANON_KEY` 改成这个 key。
4. 把这个网页发布到 GitHub Pages、Netlify 或 Vercel 之类的公网地址。
5. 重新打开发布后的网页，点击“创建云账本”，再点击“复制邀请链接”发给其他人。

注意：如果直接打开本地 `index.html`，或者使用 `localhost` 预览，复制出来的链接只能在你自己的电脑上测试，不能直接发给家人。邀请链接必须长得像 `https://你的网页地址/?ledger=...` 才能让别人打开。

没有填写 `SUPABASE_ANON_KEY` 时，应用会继续使用当前浏览器本地保存。
